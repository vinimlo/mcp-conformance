Package 2 -- npm
mcp-mock
Configurable mock server for client-side testing.
🔧 Tool Stubs
YAML-configured deterministic responses per tool
⚠ Failure Injection
Latency, error rates, timeouts, malformed responses -- chaos testing for resilience
⏯ Replay Engine
Serves recorded fixtures from mcp-conformance as deterministic sessions
📄 Session State
Multi-turn context tracking for stateful tool call sequences
🔗 Recording Pipeline
mcp-conformance records real traffic → mcp-mock replays it. One flow serves both server testing and client testing.
